El visual ya crea una carpeta para repositorio, no hay crear una nueva.
Git status reconoce la rama incluso dentro de las carpetas del directorio interno del proyecto
Cuando se guarda el fichero la extension del visual reconoce el cambio 

git status 
veo la rama en que estoy y los cambios pendientes de commit

git add "ruta y nombre del fichero"

git commit -m "mensaje"

git push , para subir los commits creados 

El flujo de trabajo mas sencillo con vscode y un navegador.

1. En el navegador abrir github y crear un nuevo repositorio.
2. En el vscode desde Source Control, clic en Open repository seleccionar el repositorio creado.
3. vscode pedira la direccion donde crear el repositorio en local y creara un workspace para empezar a funcionar.
